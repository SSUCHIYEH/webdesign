AnimeｘLeading Actress
霍爾的移動城堡－蘇菲
　　本片女主角，年齡18歲，以長女的身分代替繼母哈妮繼承 父親遺留下來的哈達帽子店，店面不大且樸素。在故事一開始顯得沒有主見，對自己的外貌也缺乏自信，對自己備受桎梏的生活感 到無奈。與霍爾互相吸引，彼此相愛。從小就很討厭蕪菁。最後恢復了原本的年歲（但是頭髮依舊為變成老婆婆後的銀髮），同時拯救了差點 死掉的霍爾並解開卡西法的契約，在重建的移動飛行城堡中過著幸福快樂的日子。
庫洛魔法使－木之本櫻
　　原本是日本友枝町小學四年級的學生，雖然幼年喪母，但在父親藤隆及哥哥桃矢的關愛中成長，是一個性格開朗又活潑的女孩。無意中打開了封印庫洛牌的書。從此，她的生活被改變。受封印之獸小可的委託，小櫻成了「庫洛魔法使」，要把擁有魔力、到處作亂的庫洛牌收服。小櫻的身邊有很多的好朋友，有知世，小狼，月城雪兔的化身——審判者月，在他們的幫助下，小櫻最終把庫洛牌一一收服，並把它們全部轉換成小櫻牌。
K!ON輕音部－平澤唯
　　一年級開學兩個星期才決定加入輕音部，加入輕音部的原因是最初錯把輕音樂當成「輕鬆的音樂」的意思而加入擁有絕對音感的天賦，但儘管到了第三年，仍對樂譜及吉他譜的讀法及術語一竅不通，也不會為吉他調音。平時為吉他調音時也只是從自己的耳朵聽辨音色，並請梓幫忙調音。由於擁有絕對音感，聽了一首歌曲一次便可以彈奏該首歌曲。懶散及天然呆的女孩。喜歡在家裏無所事事或滾來滾去。性格和善、冒有，做任何事都是做不好，但當別人氣她時，看見她傻裡傻氣的笑容，總是會原諒她。有着小孩子一樣的性格，被讚揚的話會感到非常喜悅。平時雖然甚麼事也做不好，但是本身認真專心去做的話，其效果十分驚人。
聲之形－西宮梢子
小學時轉校到男主角將也班上的聽障少女。有著先天聽覺障礙，經常被將也合夥同班上同學欺負，直到助聽器毀損賠償問題發生，因此再度轉校。在升上高中後與將也再度相會，被他想為小學時所做的事贖罪而以誠相待漸漸喜歡上將也。之後在將也的努力下開始和以前小學時的同學再次交流，但卻因為小學時種種原因令將也和大家的關係惡化。她覺得是自己的錯所以在煙火大會當晚選擇自殺，被將也所救，但同時也使將也墜樓昏迷。在將也昏迷期間夢見了將也向她道歉並準備前往另一個世界，她驚醒後馬上前往每次和將也見面的橋上，同時在醫院的將也亦醒來前往橋上，在兩人坦白心聲，將也不希望她為了以前的事情傷害自己，希望她能給他活下去的勇氣從小學時對硝子所做的過份事情中得到救贖。兩人亦終於得到救贖。
Code Geass 反叛的魯路修－C.C
在魯路修面臨生死關頭時，給予他Geass的力量救了魯路修一命。之後便寄住在魯路修家中擁有不老不死的能力，因此被視為「魔女」，曾受過刑具的虐待。其本名到現在為止還未正式出現。只有魯路修和她自己知道她的本名。她本人沒有Geass的力量。在C.C.的左胸下方和額頭都有著Geass的紅色印記。在Geass擁有者（魯路修）暴走的情況下，C.C.額頭的Geass印記有感應。C.C.與人締結契約的願望正是死去，從Geass授予者的永生地獄中擺脫，其Geass授予者的身份正是繼承於上代Geass授予者，Geass能力都對她無效。
